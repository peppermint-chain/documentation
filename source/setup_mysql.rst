============================
Setup Sample MySQL
============================

In this section we will show how to setup a sample Data-Vault using the MySQL Database.
We have instructions to setup MYSQL through two mechanisms

	a) MySQL docker image on **Docker Hub**.
	b) MySQL Community Edition for **Windows**.
